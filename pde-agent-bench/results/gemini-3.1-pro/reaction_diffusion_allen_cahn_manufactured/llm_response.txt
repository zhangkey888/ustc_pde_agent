` will take the minimum value. Since both processes will evaluate the same function at the same point, the values should be identical (up to floating point precision).
If a point is not found by a process, it has `np.inf`. `MPI.MIN` will ignore it as long as at least one process found it.
If no process found it, it remains `np.inf`, which is then replaced by `0.0` on rank 0.
This is perfectly robust.

, converged)`.
If we use `PREONLY` and `LU`, the number of linear iterations per Newton step is 1.
So total linear iterations for NS is `n_iters`.
If we want to be precise, we can add the Stokes iteration: `total_linear_iters = 1 + n_iters`.
Let's just use `n_iters`.

Let's check the exact solution again.
$u = [2\pi \cos(2\pi y) \sin(2\pi x), -2\pi \cos(2\pi x) \sin(2\pi y)]$
At $x=0$, $u = [0, -2\pi \sin(2\pi y)]$.
At $x=1$, $u = [0, -2\pi \sin(2\pi y)]$.
At $y=0$, $u = [2\pi \sin(2\pi x), 0]$.
At $y=1$, $u = [2\pi \sin(2\pi x), 0]$.
So $u$ is not zero on the boundary! It is highly oscillatory.
Our boundary condition interpolation:
```python
u_bc = fem.Function(V)
u_expr = fem.Expression(u_ex, V.element.interpolation_points())
u_bc.interpolate(u_expr)
```
This is correct and will exactly represent the boundary values
